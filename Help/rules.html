Extreme Mind is a single-player board game. Your goal is to break the secret code created by the computer. This code is made of 4 spots which can be either empty (extreme mode only) or contain a peg randomly chosen among 6 different colored pegs.

Extreme Mind offers you three different modes: easy, hard or extreme that use different code creation schemes:

  • Easy mode: Extreme Mind fills all slots with pegs and each ball has its own color.
  • Hard mode: Extreme Mind fills all slots with pegs and several pegs may share the same color.
  • Extreme mode: Extreme Mind may leave empty one or several spots and several pegs may share the same color.

You have up to 10 guesses to break the code. If you fail to break the code, Extreme Mind will display the code. When you suggest a code by pressing the "Check Code" button, the computer gives you the following information:

  • Anytime a perfect match exists for a spot (same color, same place), a black peg is shown.
  • Anytime a match exists (same color) a white peg is displayed.
