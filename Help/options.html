The menu options are as follows:
  • Game: New Game will start a new game without a warning to the user. Shortcut key is ctrl+n

  • Options: Difficulty allows you to select a different level. Shortcuts are ctrl+e for easy, ctrl+h for hard, ctrl+m for extreme. NOTE: selecting these options will start a new game and give you a warning with a chance to remain in the current game.

    Sound allows you to turn the game sounds on and off.

    Cheat allows you to view the secret code in the middle of the game. NOTE if you use the cheat you will not be permitted to enter your name in the high score list.

    View High Scores allows you to see the high score list whenever you want.

  • Help: Contents allows you to view the help file. I guess you probably found that out already or you would not be viewing it now.

    About will show you a small window showing a short message about this program.
